# Alatau Routes — сайт туристических маршрутов Алматы

Готовый статический лендинг на основе промпта пользователя.

## Файлы
- `index.html` — структура сайта и тексты
- `styles.css` — визуальный стиль
- `script.js` — фильтры маршрутов и демо-форма

## Как открыть
Откройте `index.html` в браузере.

## Что важно перед запуском
1. Подключить реальную отправку формы: WhatsApp, Telegram, email или CRM.
2. Уточнить юридические формулировки по услугам гида/трансфера.
3. Проверить актуальную доступность маршрутов, ограничения и сезонность.
4. Добавить реальные фото маршрутов и отзывы.
5. Добавить политику конфиденциальности.


## Обновление дизайна
Версия переоформлена под визуальную логику Wix-шаблона 'Коучинг': крупная журнальная типографика, теплый кремовый фон, экспертный сервисный тон, блок консультации, прямоугольные карточки и сильные CTA. Шаблон использован как референс оформления, без копирования исходных материалов Wix.
